# DataViz Studio - Latest Updates 🚀

## ✨ New Features Added

### 1. **Dark Mode by Default**
- Dark theme is now the default experience
- Sleek black and gray gradient backgrounds
- Cyan and blue accent colors for modern look

### 2. **New Ensemble Methods** 🤖
Added 4 powerful ensemble algorithms:
- **Gradient Boosting** - Sequential ensemble with learning rate control
- **AdaBoost** - Adaptive boosting with weak learners
- **Extra Trees** - Extremely randomized trees
- **Bagging Classifier** - Bootstrap aggregating

### 3. **Decision Tree Visualization** 🌳
- When using Decision Tree algorithm, you now see the actual tree structure
- Text-based tree representation showing splits and decisions
- Displayed in a terminal-style view with green text

### 4. **Enhanced UI Design** 🎨
- **Gradient backgrounds**: Dark gray to black gradients throughout
- **Glowing effects**: Cyan and blue glowing shadows on hover
- **Modern cards**: All cards have gradient borders and shadows
- **Better contrast**: Improved text visibility with cyan accents
- **Animated elements**: Smooth transitions and hover effects
